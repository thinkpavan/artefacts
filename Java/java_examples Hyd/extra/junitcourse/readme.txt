JUnit is a simple framework to write repeatable tests.

To use junit:

1) unzip junit3.8.1.zip.
2) set CLASSPATH pointing to INSTALL_DIR\junit3.8.1\junit.jar 


Our course gives just introduction to how to use junit.

for more information on junit refer to 
JUnit in Action by Vincent Massol.



Also Check httpunit framework for testing web app's. 

http://www.httpunit.org
