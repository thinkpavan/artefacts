you can download the latest version of xmlbeans from

http://xml.apache.org/xmlbeans/sourceAndBinaries/index.html

xmlbeans-current-src.zip is the current version as on july 4th 2004.


To use this you need to:

1) extract the above zip file.

2) move to the directory created when the above file is extracted (example xmlbeans-1.0.3)

3) run ant. This creates all the jar files required in build directory.

4) if you want the documentation use ant doc command.

Note: ensure that the ant is installed on your machine.

you can download ant from http://ant.apache.org/bindownload.cgi

We have used ant that is bundled with j2sdk1.4.X by javasoft for this.
